发明名称：基于元路径的意图感知虚假信息检测方法
技术领域：本发明涉及虚假信息检测领域。特别适用于如今的社交网络，使用到的技术领域包括：LLM应用开发，图神经网络，元路径多视图学习。
摘要：本发明涉及虚假信息检测领域，提出一种基于元路径的意图感知虚假信息检测方法，旨在通过社交网络中的评论数据实现虚假新闻的自动检测。该方法的核心创新在于"跨粒度意图融合"——将大语言模型的语义意图推理能力与元路径的结构意图学习能力有机结合。整体框架包括四个步骤：首先，通过大语言模型（LLM）模拟虚拟读者和作者互动，构建包含新闻节点（N）、作者节点（A）、评论节点（C）三类实体的异构评论网络；其次，利用LLM模拟领域专家显式提取写作意图和传播意图（语义层面意图）；再次，基于5条语义元路径构建多视图同构子图，通过GCN隐式学习结构层面的意图特征，并与LLM语义意图进行跨粒度融合；最后，在异构图上通过多头注意力机制（含稀疏正则化）聚合多类型边信息，实现虚假新闻的二分类检测，并引入SHAP值提供可解释性。该方法创新性地结合了显式语义意图推理与隐式结构意图学习，为虚假信息检测提供了更精细、更鲁棒的意图感知框架。
主权项：
基于元路径的意图感知虚假信息检测方法，主要包括以下过程：
（1）异构评论网络HG构建：通过大语言模型（Large Language Models, LLM）构建包含三类节点（新闻N、作者A、评论C）的异构评论网络HG。该网络中的边代表多种关系类型，包括：发布关系、评论关系、回复关系、传播意图关系、内容相似关系、点赞关系。
（2）LLM显式意图提取：将意图特征融入虚假信息检测任务。意图特征包括写作意图和传播意图。利用LLM模拟各行业专家，以专家的知识对新闻的写作意图进行预测。然后，结合第一步构建的评论网络，分析新闻传播中的特征，获取新闻的传播意图。形成语义层面意图向量[公式: I_LLM]。
（3）元路径意图感知与跨粒度融合：基于5条语义元路径从异构图中提取同构子图，通过GCN进行结构层面的意图特征学习，得到结构意图向量[公式: I_meta]。将[公式: I_LLM]与[公式: I_meta]通过门控融合机制进行跨粒度意图融合，得到最终意图表示[公式: I_final]。
（4）建立多头注意力异构图聚合网络，以统一的"多头注意力+稀疏正则化"机制聚合多类型边信息并学习注意力权重，从而检测新闻的真实性。整体流程如图所示。

根据权利要求1所述的异构评论网络HG的构建，其中步骤(1)按如下过程执行：
1.1 节点定义
异构图HG包含三类节点：
新闻节点N：待检测的新闻文章，承载新闻文本和意图特征。
作者节点A：新闻的发布者，一个作者可以发布多篇新闻，携带历史行为模式。
评论节点C：包含两类——虚拟读者评论（约10条/新闻）和作者回复（1-2条/新闻）。

1.2 评论生成策略——网络拓扑构建
定义N为新闻实体，A为作者实体，Reader表示读者集合，P表示普通读者特征向量空间，特征向量维度包括性别、年龄、收入水平及政治倾向等社会人口学属性。采用随机采样函数为每个用户[公式: ri∈R]分配特征向量[公式: pi∈P],其中[公式: i∈1,2,...,R]。采用三种评论生成策略：直接阅读路径、间接交互路径和作者回复路径。

【策略一：直接阅读路径】
设定控制参数[公式: ratio=0.6∈0,1]，通过均匀分布随机变量[公式: r~Uniform(0,1)]进行路径选择。当[公式: r<ratio]时，采用直接阅读路径。该路径为用户-新闻二分图结构，通过预训练的语言模型Qwen2-72b-instruct模拟用户阅读过程。具体而言，模型接收用户特征向量[公式: pi]与新闻内容N作为输入，输出读者评论text，其中包含原始回复内容t，新闻传播意图的揣测[公式: Is]，情感倾向emotional，对新闻的质疑等。此读者评论表示用户对新闻内容的认知与情感倾向。
[公式: text=SimulateUserReadingResponse(Qwen,N,Pi)]
其中，SimulateUserReadingResponse表示模拟用户行为的函数。

【策略二：间接交互路径】
当[公式: r≥ratio]时，采用间接交互路径。此路径引入链式传播机制，基于用户浏览行为构建多跳网络。每个读者节点[公式: ri]关联一个前序节点链[公式: Li=ri1,ri2,...,rik]，其中k表示链式长度。在读者评论生成过程中，系统同步检索并整合L的历史交互记录与新闻共同构建上下文，从而生成更符合真实情况下的读者评论text。
[公式: rj=min⁡(β∙H+(1-β)∙W)]
[公式: text=SimulateUserReadingResponse(Qwen,N,Pi,summary(Li))]
参数ratio实现了网络生成策略的动态平衡，其中直接路径侧重内容驱动的认知模拟，间接路径强调社会影响的传播效应。

【策略三：作者回复路径（新增）】
对每条新闻，LLM模拟作者角色，针对读者评论生成1-2条回复。为体现全局性，采用批量生成策略：一次性投喂同一作者的所有新闻及其下属评论，使作者回复能体现跨新闻的一致性。形式化定义如下：
[公式: text_reply=SimulateAuthorReply(Qwen,A,N,{C_1,C_2,...,C_m})]
其中，A为作者实体，N为当前新闻，[公式: {C_1,...,C_m}]为该新闻下的读者评论集合。作者回复是元路径C-AC-C（评论-作者评论-评论）成立的基础，同时丰富了异构图中作者节点与评论节点之间的连接关系。

1.3 边类型定义
在异构评论网络HG中，边被定义为以下类型：

【节点间结构关系边】
(1) 发布边[公式: E_AN]：连接作者节点A与其发布的新闻节点N，表示"作者A发布了新闻N"的关系。
[公式: E_AN={(a_i,n_j)|author(n_j)=a_i}]

(2) 评论边[公式: E_NC]：连接新闻节点与其评论节点。若评论通过直接阅读路径生成，则建立新闻N到评论C的边；若通过间接交互路径生成，则建立前序评论节点到当前评论节点的边，即形成评论间的回复链。
[公式: E_NC={(n,c)|c是n的直接评论}∪{(c_j,c_i)|c_i是对c_j的回复}]

(3) 作者回复边[公式: E_AC]：连接作者节点A与其回复的评论节点C，表示"作者A回复了评论C"的关系。
[公式: E_AC={(a_i,c_j)|a_i回复了评论c_j}]

【评论间关系边】
(4) 传播意图边[公式: EI]：基于读者对新闻的传播意图[公式: Is]分类。提取两个读者评论text1和text2的[公式: Is]，如果新闻归类为相同的传播意图，则构建传播意图边。定义意图集合为：[公式: I="信息共享","社交","政治宣传","利益宣传","制造混乱或舆论风暴","情感宣泄","谣言传播"]。传播意图边表明二者对该新闻存在相似的认知视角，具备潜在的交互价值。
[公式: Is=ExtractSpreadingIntent(text)]
[公式: EI_ij=1 if I_i==I_j, 0 else]

(5) 相似边[公式: ES]：用于衡量读者评论text的相似性。当不同的读者有类似表达时，彼此的评论有更多值得学习的地方。在此使用TF-IDF余弦相似度C，通过量化词语的重要性并比较文本向量的相似性，实现对文本内容的有效分析。如果C>0.5，则在两条评论节点之间构建相似边。
[公式: CosineSimilarity=texti•textj/(|texti|*|textj|)]
[公式: ES_ij=1 if CosineSimilarity(c_i,c_j)>0.5, 0 else]

(6) 点赞边[公式: EA]：进一步模拟用户在网络中的互动行为。若某一节点[公式: c_i]表达对另一节点[公式: c_j]的text支持或认同，两者之间建立一条点赞边。这种点赞行为不仅反映了用户之间的态度一致性，也有助于揭示节点间潜在的情感共鸣和社会影响力结构。
[公式: EA_ij=1 if GenerateLikeBehavior(c_i,c_j)=true, 0 else]

1.4 异构图形式化定义
异构评论网络HG(V,E,R)的定义如下：
[公式: HG(V,E,R)={R={'发布','评论','回复','意图','相似','点赞'}, V={N,A,C} and |V|=n, E⊆{(v_i,v_j,e_ε)|v_i,v_j∈V,ε∈R}}]
异构边索引集合[公式: ε={E_AN,E_NC,E_AC,EI,ES,EA}]，其中每个子集对应一种特定类型的边关系，共同构成多个子图结构。

根据权利要求1所述的基于元路径的意图感知虚假信息检测方法，其中步骤(2)按如下过程执行：
为深入解析新闻文本中所承载的多维意图结构，本文将意图划分为两个主要维度：写作意图[公式: It]与传播意图[公式: Is]。这两个维度分别反映了作者在文本生成阶段的主观目的以及信息在网络传播过程中读者的意图揣测。

2.1 写作意图[公式: It]提取
为更好刻画写作意图[公式: It]，构建了一个由多个专业领域组成的专家读者集合[公式: E={e1,e2,...,en}]，具体包括以下核心领域（从全覆盖式精简为核心视角，在降低计算成本的同时保留关键判断能力）：新闻评论专家、心理学专家、社会学专家、数据科学家。每个专家[公式: ei]对应一个特定的知识领域[公式: Di]，其专业能力通过该领域的知识库[公式: Ki]进行建模。在此基础上，采用具有相应领域微调的预训练语言模型作为模拟专家行为的代理。通过设计，选取三个维度来体现写作意图[公式: It]，分别是<信念B、欲望D、计划P>。其中信念意图表示新闻的主要目标是否是为了说服读者相信某个观点。欲望意图表示新闻是否试图激发读者对某件事情的渴望或厌恶。计划意图表明新闻是否试图影响读者的行为或让他们采取某些特定的行为。通过对每位专家模型设置一组特定的二分类问题，实现对上述三类子意图维度的识别与量化。
最后，系统汇总所有专家模型输出的意图预测结果，并根据各领域专家的专业相关性设定加权系数[公式: ωi]，对意图预测值进行加权求和，得到最终的写作意图表示：
[公式: It=Σ(i=1~n)ωi⋅PredictExpertWritingIntent(N,ei,Di,Ki)]
[公式: B=InferBelief(N,ei,Di,Ki)]
[公式: D=DetermineDesire(N,ei,Di,Ki)]
[公式: P=PlanWriting(N,ei,Di,Ki)]
[公式: PredictExpertWritingIntent(N,ei,Di,Ki)=(B,D,P)]
其中,PredictExpertWritingIntent()是主函数，表示第i位专家模型对新闻文本N的意图预测，通过预训练模型给出判断，权重[公式: ωi]反映该专家在当前任务中的相对重要性。

2.2 传播意图[公式: Is]提取
在分析新闻评论网络中读者之间的互动行为基础上，进一步定义传播意图[公式: Is]，用以刻画用户在参与信息扩散过程中所表现出的立场倾向与交互特征。具体而言，构建了一个基于评论序列的动态图结构HG，该图不仅保留原始新闻评论的文本信息，还整合了用户间的多模态互动行为，包括点赞、回复等社交反馈机制。在此基础上，将构建好的图结构HG与新闻文本一同输入大型语言模型（Large Language Model, LLM），通过其强大的语义理解和上下文建模能力，挖掘评论序列中隐含的用户立场演变、情绪波动以及观点演化趋势。LLM的输出用于生成传播意图表示向量[公式: Is]，其形式化定义如下：
[公式: Is=get_Intent(LLM,HG)]

2.3 语义意图向量
[公式: I_LLM=(It,Is)]
将写作意图和传播意图拼接，形成语义层面的意图表示。此向量将在步骤(3)中与元路径结构意图进行跨粒度融合。

本步骤是整合方案的核心创新——通过元路径从异构图结构中隐式学习不同维度的意图特征，并与步骤(2)中LLM提取的语义意图进行跨粒度融合，形成更加全面、鲁棒的意图表示。

3.1 语义元路径定义
元路径（Meta-path）是异构信息网络中连接两个节点的路径模式，由一系列节点类型和边类型交替组成。本方法定义5条具有明确意图语义的元路径，每条元路径对应一种特定的意图维度：

元路径1：N-A-N（新闻→作者→新闻），捕获的意图语义为作者的写作意图，包括写作方式、偏好、立场等。其直觉含义为：同一作者发布的不同新闻之间共享作者的写作风格和意图模式。

元路径2：N-C-N（新闻→评论→新闻），捕获的意图语义为传播对写作意图的效果。其直觉含义为：通过评论将不同新闻关联，反映受众对新闻写作意图的反馈模式。

元路径3：N-C-A（新闻→评论→作者），捕获的意图语义为作者写作意图是否被读者支持。其直觉含义为：读者评论与作者之间的关联模式反映了受众对作者意图的认同程度。

元路径4：C-AC-C（评论→作者评论→评论），捕获的意图语义为作者与读者互动中的传播意图与写作意图。其直觉含义为：通过作者回复将不同读者的评论关联，揭示作者在互动中展现的传播策略。

元路径5：C-C-C（评论→评论→评论），捕获的意图语义为读者间互动体现的传播意图。其直觉含义为：评论间的回复链和互动模式反映了信息在读者群体中的传播扩散方式。

3.2 元路径同构子图构建
从异构图HG中，按每条元路径[公式: Φ_k(k=1,2,...,5)]提取对应的同构子图[公式: G_k=(V_k,E_k)]。具体而言，对于元路径[公式: Φ_k]，若两个同类型节点之间存在符合该元路径模式的连接路径，则在同构子图中建立一条边。

【元路径1子图 G_1（N-A-N，新闻同构图）】
映射规则：若两条新闻[公式: n_i]和[公式: n_j]由同一作者[公式: a]发布，即[公式: author(n_i)=author(n_j)=a]，则在[公式: G_1]中建立边[公式: (n_i,n_j)]。
边权设计：为防止高产作者主导图结构，采用以下策略：
  (a) 同作者连边：[公式: E_1={(n_i,n_j)|author(n_i)=author(n_j)}]
  (b) top-k邻居保留：每个新闻节点仅保留与其语义最相近的k个同作者新闻邻居
  (c) 边权归一化：[公式: w_ij=sim(n_i,n_j)/Σ_{n_k∈topk(n_i)}sim(n_i,n_k)]
  (d) 自环添加：[公式: E_1=E_1∪{(n_i,n_i)|n_i∈V_1}]，保留节点自身特征

【元路径2子图 G_2（N-C-N，受众反馈视图）】
映射规则：若评论[公式: c]同时关联新闻[公式: n_i]和新闻[公式: n_j]（例如评论内容涉及多条新闻），则建立边[公式: (n_i,n_j)]。实践中采用二部图结构，仅执行C→N方向的聚合。
采样策略：(a) top-k选择语义最相关的评论；(b) 余弦相似度筛选减少噪音；(c) MMR（Maximal Marginal Relevance）计算保证评论多样性。

【元路径3子图 G_3（N-C-A，支持度视图）】
映射规则：连接新闻N、评论C、作者A三类节点，捕获读者对特定作者新闻的反应模式。若评论[公式: c]属于新闻[公式: n]，且[公式: n]由作者[公式: a]发布，则形成路径[公式: n→c→a]。

【元路径4子图 G_4（C-AC-C，互动传播视图）】
映射规则：若作者[公式: a]的回复评论[公式: c_a]分别关联了读者评论[公式: c_i]和[公式: c_j]（即作者同时回复了这两条评论），则在[公式: G_4]中建立边[公式: (c_i,c_j)]。此子图的构建依赖于步骤(1)中生成的作者回复（1-2条/新闻）。

【元路径5子图 G_5（C-C-C，读者传播视图）】
映射规则：基于评论间的直接互动关系（回复链、相似关系），若评论[公式: c_i]与[公式: c_j]之间存在中间评论[公式: c_m]使得[公式: c_i→c_m→c_j]，则在[公式: G_5]中建立边[公式: (c_i,c_j)]。

3.3 元路径GCN编码
每条元路径子图[公式: G_k]各使用一个独立的图卷积网络（Graph Convolutional Network, GCN）进行特征学习。GCN通过消息传递机制聚合邻居信息，学习图结构中的意图模式。

对于第k条元路径子图[公式: G_k=(V_k,E_k)]，设节点初始特征矩阵为[公式: X_k∈R^(|V_k|×d_in)]，其中[公式: |V_k|]为节点数量，[公式: d_in]为输入特征维度。GCN的逐层传播规则定义如下：

[公式: H_k^(l+1)=σ(D̃_k^(-1/2)Ã_kD̃_k^(-1/2)H_k^(l)W_k^(l))]

其中：
  [公式: Ã_k=A_k+I]为添加自环后的邻接矩阵，[公式: A_k]为子图[公式: G_k]的邻接矩阵，[公式: I]为单位矩阵
  [公式: D̃_k]为[公式: Ã_k]的度矩阵，[公式: D̃_k,ii=Σ_jÃ_k,ij]
  [公式: W_k^(l)∈R^(d_l×d_{l+1})]为第l层的可学习权重矩阵
  [公式: H_k^(0)=X_k]为初始输入
  [公式: σ]为ReLU激活函数

经过L层GCN传播后，得到第k条元路径视图下的节点特征矩阵[公式: H_k^(L)∈R^(|V_k|×d_out)]。对于新闻节点[公式: n_i]，其在第k条元路径视图下的意图特征向量为：
[公式: h_k(n_i)=H_k^(L)[i]∈R^(d_out)]

特别地，对于元路径1（N-A-N），采用attention pooling作为编码策略，计算同作者新闻对目标新闻的注意力权重：
[公式: α_ij=exp(LeakyReLU(a_1^T[W_1·h(n_i)||W_1·h(n_j)]))/Σ_{n_k∈topk(n_i)}exp(LeakyReLU(a_1^T[W_1·h(n_i)||W_1·h(n_k)]))]
[公式: h_1(n_i)=Σ_{n_j∈topk(n_i)}α_ij·h(n_j)]

对于元路径2（N-C-N），采用特殊的attention设计，以新闻作为query计算评论对新闻的重要程度：
[公式: α_cj=exp(LeakyReLU(a_2^T[W_q·h(n_i)||W_v·h(c_j)]))/Σ_{c_k∈C(n_i)}exp(LeakyReLU(a_2^T[W_q·h(n_i)||W_v·h(c_k)]))]
[公式: h_2(n_i)=Σ_{c_j∈C(n_i)}α_cj·h(c_j)]

其中[公式: W_q]为query变换矩阵（新闻侧），[公式: W_v]为value变换矩阵（评论侧），[公式: C(n_i)]为新闻[公式: n_i]的评论集合。

3.4 多头注意力融合
使用5头注意力机制对应5条元路径，学习各视图的重要性权重。具体而言，对于新闻节点[公式: n_i]，其在5条元路径下分别获得意图特征向量[公式: {h_1(n_i),h_2(n_i),...,h_5(n_i)}]。通过注意力机制学习各元路径的贡献权重：

首先计算每条元路径的注意力得分：
[公式: e_k=q^T·tanh(W_att·h_k(n_i)+b_att)]

其中[公式: W_att∈R^(d_att×d_out)]为注意力变换矩阵，[公式: q∈R^(d_att)]为注意力查询向量，[公式: b_att]为偏置项。

然后通过softmax归一化得到注意力权重：
[公式: α_k=exp(e_k)/Σ_{j=1}^{5}exp(e_j)]

最终的结构意图向量通过加权求和得到：
[公式: I_meta(n_i)=Σ_{k=1}^{5}α_k·h_k(n_i)]

其中[公式: I_meta(n_i)∈R^(d_out)]即为新闻节点[公式: n_i]在图结构层面的意图特征向量，融合了5种不同维度的意图语义信息。

3.5 跨粒度意图融合
将步骤(2)中LLM提取的语义意图（粗粒度、内容驱动）与元路径学习的结构意图（细粒度、关系驱动）进行融合。首先对语义意图进行向量化处理：
[公式: I_LLM_embed=Embed(It);Embed(Is)]

其中[公式: Embed]为预训练语言模型的嵌入函数，分号表示向量拼接操作。

采用门控融合机制（Gated Fusion）实现跨粒度融合，该机制通过可学习的门控参数自动调节两种意图的融合比例：

[公式: gate=σ(W_g·[I_LLM_embed;I_meta]+b_g)]
[公式: I_final=gate⊙I_LLM_embed+(1-gate)⊙I_meta]

其中[公式: W_g∈R^(d×2d)]为门控权重矩阵，[公式: b_g∈R^d]为偏置项，[公式: σ]为Sigmoid激活函数，[公式: ⊙]为逐元素乘法（Hadamard积）。gate的每个维度取值在[0,1]之间，自适应地控制每个特征维度上语义意图与结构意图的贡献比例。

跨粒度融合的价值在于：LLM语义意图基于单篇新闻内容即可判断，不依赖图结构的完整性；元路径结构意图从图拓扑中发现隐含模式，提供数据驱动的验证。两者互相印证，例如LLM判断"作者有误导意图"，而N-A-N路径显示该作者历史上确实多次发布虚假信息，两者共同增强了判断的可信度。

根据权利要求1所述的基于元路径的意图感知虚假信息检测方法，其中步骤(4)按如下过程执行：

4.1 文本嵌入层
本网络采用预训练的语言模型Qwen3-Embedding作为文本特征提取的核心模块。该模型通过大规模语料库的无监督预训练与监督微调相结合的方式，能够将离散文本序列映射为连续且具有语义关联的低维向量空间表示。

具体而言，获取每个节点的文本表示Text=[t1,t2,...,tn]：新闻节点获取新闻文本，评论节点获取评论文本，作者节点获取作者历史行为描述。首先进行标准化编码，生成初始语义特征矩阵[公式: S∈R^(n×ds)]，其中n为节点数量，[公式: ds]为初始语义维度。
[公式: S=[Embed(t1),Embed(t2),...,Embed(tn)]]

将跨粒度意图向量[公式: I_final]与新闻文本特征进行融合。具体而言，将写作意图[公式: It]与传播意图[公式: Is]进行向量化拼接，结合元路径结构意图，形成意图增强的新闻节点表示：
[公式: S_NI=[I_final;Embed(N)]]

最终构建初始特征矩阵：
[公式: X=Concat(S_NI,S)∈R^(n×d_in)]
将意图评论特征矩阵作为嵌入层的输出，用于后续GNN层处理。

4.2 GNN层：多头注意力异构聚合
尽管当前的异构图神经网络已广泛采用多级注意力机制或多权重矩阵策略，以捕捉并区分不同类型邻域对节点表示的影响，但这些方法仍存在一个关键局限性：它们通常假设同一层中所有节点对邻域类型的需求是统一的，从而忽略了不同节点在信息聚合过程中可能具有的个性化邻域需求。

为解决这一问题，本研究提出统一的"多头注意力+稀疏正则化"异构聚合机制。该机制采用注意力权重软分配取代硬选择策略，同时通过稀疏正则化约束鼓励模型自动学习各边类型的重要性分布——若某种边类型对特定节点构成噪音，其权重将自动趋近于零（等效于硬选择的隔绝效果）；若仍有微弱贡献，模型也不会完全丢弃。

具体过程如下：

(1) 按边类型分别聚合邻居信息：
对每种边类型[公式: d∈{评论边,意图边,相似边,点赞边}]，定义节点v在边类型d下的邻居集合：
[公式: N_d(v)={u|(v,u,etype=d)∈HG,d∈ε}]

对每种边类型使用GATv2计算注意力系数。将节点v和其边类型d下的邻居节点u通过GATv2计算注意力系数[公式: α_vu^d]，表示节点v对其邻居节点u在该边类型下的"关注程度"：
[公式: α_vu^d=exp(LeakyReLU(a_d^T[W_d·h_v^(l-1)||W_d·h_u^(l-1)]))/Σ_{k∈N_d(v)}exp(LeakyReLU(a_d^T[W_d·h_v^(l-1)||W_d·h_k^(l-1)]))]

其中[公式: W_d∈R^(d_out×d_in)]为边类型d对应的线性变换矩阵，[公式: a_d∈R^(2d_out)]为边类型d的注意力参数向量，||为拼接操作。

根据注意力系数对邻居表示进行加权求和，得到边类型d下的聚合结果：
[公式: h_d(v)=Σ_{u∈N_d(v)}α_vu^d·h_u^(l-1)]

(2) 多头注意力跨边类型融合 + 稀疏正则化：
对4种边类型的聚合结果进行注意力加权融合。首先计算各边类型的重要性得分：
[公式: e_d=p^T·tanh(W_β·h_d(v)+b_β)]

其中[公式: W_β∈R^(d_β×d_out)]为变换矩阵，[公式: p∈R^(d_β)]为查询向量。

通过softmax归一化得到边类型权重：
[公式: β_d=exp(e_d)/Σ_{d'=1}^{4}exp(e_{d'})]

引入L1稀疏正则化项，鼓励部分边类型权重趋近于零：
[公式: L_sparse=λ·Σ_{d=1}^{4}|β_d|]

其中[公式: λ]为稀疏正则化系数，控制稀疏化的强度。

最终更新节点v的特征表示：
[公式: h_v^(l)=σ(Σ_{d=1}^{4}β_d·h_d(v))]

其中[公式: σ]为ReLU激活函数。节点v包含自环边以保留节点v自身的特征信息。

(3) 特征变换增强表达能力：
在每层GNN中，通过线性变换增强节点特征的表达能力：
[公式: X^(l)=LeakyReLU(W^(l)·X^(l))]
其中[公式: W^(l)]为第l层可学习权重矩阵。

在完成多头注意力异构聚合的节点特征融合后，得到整个GNN层的输出：特征矩阵[公式: X^(out)∈R^(N×d_out)]，其中[公式: h_N^(out)]为新闻节点特征表示。

4.3 分类层
通过特征融合与非线性映射，将GNN提取的融合了丰富特征的新闻节点N，通过参数化的线性变换与激活函数，逐步抽象出高维判别特征，最终输出类别概率分布。具体而言，通过一个线性层将新闻节点特征表示映射到512维度，再通过ReLU激活函数进行激活，再通过Dropout正则化机制随机丢弃部分激活值，最后映射至辨别性二维空间，从而为新闻真实性的二分类任务（真/假）提供决策边界。分类层的前向传播过程可形式化为：
[公式: MLP(h_N^(out))=Softmax(W2·Dropout(ReLU(W1·h_N^(out)+b1))+b2)]
其中W为权重矩阵，b为偏置项。

总损失函数定义为交叉熵损失与稀疏正则化的组合：
[公式: L_total=L_CE+L_sparse=-Σ_{i=1}^{N}[y_i·log(ŷ_i)+(1-y_i)·log(1-ŷ_i)]+λ·Σ_{d=1}^{4}|β_d|]
其中[公式: y_i]为真实标签，[公式: ŷ_i]为模型预测概率。

4.4 可解释性增强
为了提高虚假信息检测模型的透明度和可信度，引入SHAP值来进行特征重要性分析。通过可解释性增强，可以得知新闻的文本特征、写作意图特征、传播意图特征、元路径结构特征、评论特征对预测结果的影响程度。通过这种方式，能识别出哪些特征在区分真实与虚假新闻中起到了最关键的作用，还能提供一种局部解释机制，帮助理解单个预测背后的逻辑。
[公式: θ_i=Σ_{S⊆F\{i}}[|S|!(|F|-|S|-1)!/|F|!]·[f_x(S∪{i})-f_x(S)]]
其中S代表不包含特征i的任意特征子集，[公式: f_x(S)]表示仅使用特征子集S时模型的预测值，分数[公式: |S|!·(|F|-|S|-1)!/|F|!]被称为权重因子，确保每个特征组合被公平地考虑。得到每一个特征的[公式: θ_i]，并为新闻真实性分类做出解释。

示例：一条新闻被分类为假新闻（0），各SHAP值：文本特征+0.2，写作意图特征+0.35，传播意图特征+0.15，元路径特征+0.20，评论特征0.10。最关键的因素是写作意图，意图"引用权威但缺乏具体来源"（+0.35），这类手法常用于伪装成权威报道以欺骗读者；元路径验证：该作者历史新闻中有多篇被标记为虚假（N-A-N路径确认，+0.20）；再加上标题中的夸张数字和情绪化表达（+0.2），综合考虑分类为假新闻。

算法 2 元路径意图感知与跨粒度融合
数据：异构评论网络HG(V,E,R)，5条元路径定义{Φ_1,...,Φ_5}，LLM语义意图向量I_LLM
输出：跨粒度融合意图向量I_final

1:  // 阶段一：元路径同构子图构建
2:  for k=1,...,5 do
3:      G_k = ExtractMetaPathSubgraph(HG, Φ_k)
4:      X_k = GetNodeFeatures(G_k)
5:  end for

6:  // 阶段二：元路径GCN编码
7:  for k=1,...,5 do
8:      H_k^(0) = X_k
9:      for l=0,...,L-1 do
10:         H_k^(l+1) = σ(D̃_k^(-1/2)·Ã_k·D̃_k^(-1/2)·H_k^(l)·W_k^(l))
11:     end for
12:     h_k = H_k^(L)    // 第k条元路径的意图特征
13: end for

14: // 阶段三：多头注意力融合
15: for k=1,...,5 do
16:     e_k = q^T · tanh(W_att · h_k + b_att)
17: end for
18: {α_1,...,α_5} = Softmax({e_1,...,e_5})
19: I_meta = Σ_{k=1}^{5} α_k · h_k

20: // 阶段四：跨粒度融合
21: I_LLM_embed = [Embed(It) ; Embed(Is)]
22: gate = σ(W_g · [I_LLM_embed ; I_meta] + b_g)
23: I_final = gate ⊙ I_LLM_embed + (1-gate) ⊙ I_meta

24: Return I_final


算法 3 多头注意力异构图聚合
数据：特征矩阵X∈R^(n×d_in)，异构图HG，边类型集合ε={评论边,意图边,相似边,点赞边}，稀疏系数λ
输出：新闻节点特征h_N^(out)

1:  // 文本嵌入层
2:  S = [Embed(t1), Embed(t2), ..., Embed(tn)]
3:  S_NI = [I_final ; Embed(N)]
4:  X = Concat(S_NI, S)

5:  // GNN层
6:  for l=1,...,L do
7:      for each node v in V do
8:          // 按边类型分别聚合
9:          for each edge type d in ε do
10:             for each neighbor u in N_d(v) do
11:                 α_vu^d = GATv2Attention(h_v^(l-1), h_u^(l-1), W_d, a_d)
12:             end for
13:             h_d(v) = Σ_{u∈N_d(v)} α_vu^d · h_u^(l-1)
14:         end for
15:
16:         // 跨边类型注意力融合
17:         for each edge type d in ε do
18:             e_d = p^T · tanh(W_β · h_d(v) + b_β)
19:         end for
20:         {β_1,...,β_4} = Softmax({e_1,...,e_4})
21:
22:         // 稀疏约束
23:         L_sparse += λ · Σ_{d=1}^{4} |β_d|
24:
25:         // 更新节点特征
26:         h_v^(l) = σ(Σ_{d=1}^{4} β_d · h_d(v))
27:     end for
28: end for

29: // 分类层
30: h_N^(out) = X^(L)[N]    // 提取新闻节点特征
31: ŷ = Softmax(W2 · Dropout(ReLU(W1 · h_N^(out) + b1)) + b2)
32: L_total = L_CE(y, ŷ) + L_sparse

33: Return ŷ, L_total


创新点总结：
1.【跨粒度意图融合机制】首次将LLM语义意图推理（粗粒度、内容驱动）与元路径结构意图学习（细粒度、关系驱动）有机融合。两者互补——LLM擅长理解单篇新闻的语义意图，元路径擅长从图结构中发现跨新闻的隐含意图模式。通过门控融合机制自适应地调节两种意图的贡献比例。

2.【语义元路径意图感知】设计5条具有明确意图语义的元路径（N-A-N、N-C-N、N-C-A、C-AC-C、C-C-C），分别捕获写作偏好、传播效果、支持度、互动意图、传播扩散五个维度的意图特征，实现意图的细粒度解耦与学习。

3.【统一的注意力融合机制】提出"多头注意力+稀疏正则化"的统一框架，同时处理元路径视图融合和异构边类型融合。模型通过稀疏约束自动学习权重分配——噪音源的权重自动趋近于零（等效于硬选择），有效信息源保留适当权重。避免了硬选择策略的信息损失问题，无需人为设定决策边界。

4.【LLM驱动的三实体评论网络】构建包含新闻（N）、作者（A）、评论（C）三类节点的异构图，在虚拟读者评论的基础上引入作者回复机制，丰富了图结构的互动维度，为元路径分析提供了必要的图结构基础。

数据集与实验设计：
数据集选择：LIAR
选用LIAR数据集，原因：(1)无自带评论——可以用LLM自主构建评论网络，避免真实评论的噪音干扰；(2)有作者信息——支持元路径N-A-N、N-C-A等需要作者节点的路径；(3)足够权威——学术界广泛使用的虚假信息检测基准数据集。

  - 为体现全局性：一次性投喂同一作者的所有新闻
  - 文本向量化：Qwen3-Embedding，1024维
  - 元路径子图：预先构建并保存为.pt文件
